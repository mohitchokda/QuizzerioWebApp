3. For 16-bit compiler allowable range for integer constants is ________?

  • A. -3.4e38 to 3.4e38
  • B. -32767 to 32768
  • C. -32668 to 32667
  • D. -32768 to 32767

Answer:D

In a 16 Bit C compiler we have 2 bytes to store an integer, and 1 byte for a character. For unsigned integers the range is 0 to 65535. For signed integers the range is -32768 to 32767. For unsigned character, 0 to 255

10. What will be printed after execution of the following program code?


main() { printf("\\nab"); printf("\\bsi"); printf("\\rha"); }

  • A.absiha
  • B. asiha
  • C.haasi
  • D.hai

Answer:D

\\n - newline - printf("\\nab"); - Prints 'ab' \\b - backspace - printf("\\bsi"); - firstly '\\b' removes 'b' from 'ab ' and then prints 'si'. So after execution of printf("\\bsi"); it is 'asi'. \\r - linefeed - printf("\\rha"); - Now here '\\r' moves the cursor to the start of the current line and then override 'asi' to 'hai' .
